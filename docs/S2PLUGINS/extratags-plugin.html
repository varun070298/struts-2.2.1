Overview

The ExtraTags plugin provides a set of additional tags that complement the core UI tags. These tags allow a user to customize the templates for standard html elements that are not included in the core tag.

Features

  • Body UI tag including onload and onunload javascript events and customizable ftl template
  • Html UI tag including customizable ftl template
  • Head UI tag including customizable ftl template

Installation

This plugin can be installed by copying the plugin jar into your application's /WEB-INF/lib directory.
